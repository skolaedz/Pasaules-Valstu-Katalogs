Interaktīva tīmekļa lietotne, kas ļauj lietotājiem pētīt informāciju par pasaules valstīm, izveidota izmantojot React, TypeScript, Axios, Tailwind CSS.
Lietotne komunicē ar publisko API, kas nodrošina ērtu un vizuāli pievilcīgu saskarsmi, lai piekļūtu aktuālajai statistikai par valstīm, to iedzīvotāju skaitu, reģioniem un karogiem.

Palaižot programmu, terminālā jāievada:
1) npm install
2) npm start
3) Atvērt: `http://localhost:3000`
